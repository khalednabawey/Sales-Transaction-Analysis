## Sales Transaction Analysis

This project focuses on analyzing sales transaction data to uncover actionable insights that can help businesses make data-driven decisions. The analysis includes sales performance trends, customer behavior, product performance, and revenue analysis using advanced data analytics techniques.

### Objectives
- Analyze sales trends over time.
- Identify top-performing products and categories.
- Understand customer purchasing behavior.
- Visualize KPIs like revenue, profit, and order counts.
- Build an interactive Power BI dashboard connected to the SQL database.

### Tools Used 
- PostgreSQL
- PowerBI
